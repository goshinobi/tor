# tor 
This is tor proxy controller package.

## what is Tor?
* [Wikipedia - Tor](https://en.wikipedia.org/wiki/Tor)

## install tor_multi
```
$ go install github.com/goshinobi/tor/cmd/tor_multi
```

## install tor
### OS X
```
$ brew install tor
```

## API
### list proxy
```
/list
```

### add proxy
```
/add
```

### kill all proxy
```
/killAll
```
